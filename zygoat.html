Mission

The future of data is DNA. It's size and versatility would drastically reduce the cost and space that server operation would require, which is really exaaggerated in this video. The goal of this research project is to create a server that will store data in DNA, and to use these servers to deliver high speed Internet connections at next to nothing by storing large parts of the web locally.

Progress

The first thing we have to do in order to build a DNA server is decrease the cost of writing DNA . This will be done by building a PCR system that not only copies DNA, but also writes it. It would do so by copying and ligating small strands.
